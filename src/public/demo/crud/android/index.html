CRUD Demo App - Vaskit

Welcome! This demo app shows how you can use VASKIT to create, remove, update and delete records.

Things you will need:

  • 1) A VASKIT account. Go here to get one.
  • 2) Your DEVELOPER KEY. You get this from your Dashboard
  • 3) From your Dashboard, create a table called employees
  • You can find this code in our repository
  • Click here and learn about users and records security to make them private or accessible to certain users only. Learn how to send PUSH messages and use SOCKET communication.
Android

Please not this is a very simple example and it must be used only as a reference.

This exmple is developed using Java and Volley as the library for sending requests to VASKIT.
